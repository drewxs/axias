# Axias

A platform for API development.
<br><br>

### Usage

Simply visit [Axias](https://axias.vercel.app/).
<br><br>

### Development

To clone this repo on your local machine:

```sh
git clone https://github.com/kito0/axias.git
```

Install dependencies

```sh
npm i
```

Start dev client:

```sh
npm start
```
